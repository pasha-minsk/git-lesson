  • git clon путь к репозиторию (с githab странички )
  • git status - статус текущего локального репазитория
  • git add "путь к файлу название файла" - добавляет новый файл в область видимости git, под его контроль на нашем компе
  • git commit -am "added new file index.html - тоесть осмысленный коментарий того что я поменял"
  • git push - отправить все закомиченные изменения в репозиторий на сервер
  • git pull - получить все закомиченные изменения с репозитория сервера
